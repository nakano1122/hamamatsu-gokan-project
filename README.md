## アプリ名
**浜松de五感を満たすおでかけ提案サービス「どこ行く？」**

## アプリ概要
- 公式LINEで浜松市の観光スポットを推薦
- 公式LINE内でbotと会話をする中でユーザの好みを把握し、パーソナライズ

## システム概要
- LINEの公式LINEでユーザとの対話を実現
- LINEが提供するAPIであるMessaging APIを使用
- 観光スポットの検索は、チームメンバーがこれとは独立したAPIを開発した
